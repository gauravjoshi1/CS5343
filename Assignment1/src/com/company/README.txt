Both the java files BubbleSort.java and TertiarySearch.java have their own main methods which will be used to run test cases.
Compile the code on the terminal as follows:

 javac BubbleSort.java (generates class file for BubbleSort)
 javac TertiarySearch.java (generates class file for TertiarySearch)

 Run the compiled class files

 java BubbleSort
 java TertiarySearch
